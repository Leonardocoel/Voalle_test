## Getting Started

### Setup

- Clone the repository and switch to it

  ```bash
  git clone https://github.com/Leonardocoel/Voalle_test && cd Voalle_test/src
  ```

- Install packages

  ```bash
  dotnet restore
  ```

- Run the app

  ```bash
  dotnet run --project Console/Console.csproj
  ```
